The Subtitles Presenter For Live Performance

Developed C# using WPF a personal use - present subtitles for the forein language songs in my singing recital.

How to use:

First you put your subtitles in a txt file while the slides are seperated in new lines. See example file here.
Than you just load the subtitles and click "Show". You can go back and forth using the arrow or the mouse wheel.

Use "more option" to adjust the location and size of the subtitles.

The Arabic Verb Builder

Developed in C# using WinForm.

I developed it during my Arabic language study in order to help myselft and other learn the complex verb structure of the Arabic language. This software implements all the special grammar rules and enable the using to "build" a verb using the root, and the desire mood (tense) and pronoun.

WARNING: this software is not properly tested so it might contain bugs. do not count on it to be perfect. If you find an error contact me at: shachar.harshuv@gmail.com.
